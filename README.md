#  Delícias do Sabor

O **Delícias do Sabor** é um projeto de site desenvolvido com o objetivo de apresentar receitas e pratos culinários de forma organizada e visualmente atrativa.

Este projeto foi criado para praticar conceitos de **desenvolvimento front-end**, incluindo estruturação de páginas HTML, manipulação do DOM com JavaScript e criação de interfaces simples e funcionais.

---

##  Demonstração do Projeto

<img src="deliciasDoSaborInterface.png"/>

Este projeto simula um site de culinária onde os usuários podem visualizar receitas, pratos e conteúdos relacionados à gastronomia.

---

##  Objetivo do Projeto

O objetivo deste projeto é:

* Praticar desenvolvimento web
* Criar uma interface de site de receitas
* Aprender manipulação do DOM com JavaScript
* Trabalhar com organização de páginas HTML
* Desenvolver habilidades em front-end
* Criar um projeto para portfólio

---

##  Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Manipulação do DOM
* Estruturação de páginas web
